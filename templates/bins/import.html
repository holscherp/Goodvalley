Import Bins from Excel
Expected Format — Warehouse Export

The system reads your standard warehouse Excel file automatically. It searches for a header row containing TARJA and reads these columns:

Column Header name What it maps to
BTARJABin ID
CNETOWeight (kg)
MPRODUCTORProducer name
PSERIECaliber (e.g. 50/60) — N/A is accepted
QSECADODrying method (CANCHA, HORNO, etc.)
  • SECADO accepted values: CANCHA (field), HORNO (oven), OTRO (other)
  • SERIE can be N/A — caliber can be updated manually later
  • Rows with a duplicate TARJA are skipped automatically
  • Title rows and empty rows before the header are ignored
