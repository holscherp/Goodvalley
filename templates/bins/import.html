Required Excel Format

The first row must be a header. Data starts on row 2. Columns must appear in this exact order:

Column A Column B Column C Column D Column E Column F
Bin ID Producer Weight (kg) Drying Method Caliber Notes
BIN-001 Smith Farm 320.5 Oven Drying 50/60 optional
  • Drying Method must be: Oven Drying, Field Drying, or Other
  • Caliber must be in XX/YY format (e.g. 50/60)
  • Rows with a duplicate Bin ID are skipped automatically
  • Fully empty rows are ignored
